Week 2: Understanding how the table element works.

The table element in HTML is used to display data in a structured, tabular format. It consists of rows (tr), columns with headers (th), and data cells (td). Tables help organize information neatly, making it easy to compare values.

Week 3:Html Forms and other html elements which are required of me to know

Forms in HTML (form) allow users to submit data through various input fields like text boxes (input), dropdowns (select), and buttons (button). Each input field is typically associated with a (label)
to improve accessibility and usability. The (div) element is a block-level container used to group elements for styling or layout purposes. In contrast, block elements (e.g., p, h1, div) take up the full
width of their parent container, whereas inline elements (e.g., span, a, strong) only take up as much space as needed. Proper structuring of forms and elements enhances user experience and page organization.
